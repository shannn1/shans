# Shan's Space

个人空间 - 记录阅读、观影与旅行的时光

## 功能

- 📚 读书记录：记录读过的书籍，包括封面、笔记、评分
- 🎬 观影记录：记录看过的电影，包括海报、感想、评分
- ✈️ 旅行见闻：记录旅行经历，包括照片、地点、见闻

## 技术

- 纯前端实现（HTML + CSS + JavaScript）
- 数据存储在浏览器 localStorage
- 响应式设计，支持移动端和桌面端

## 本地运行

直接在浏览器中打开 `index.html` 即可。

## 部署

使用 GitHub Pages 部署，访问地址：`https://YOUR_USERNAME.github.io/shan-space/`
